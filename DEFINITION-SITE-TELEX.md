# 🎬 Définition Complète du Site Web TELEX

## 📋 **Définition Générale**

### **🎯 TELEX : Télévision Étudiante Excellence**
TELEX est une **plateforme web complète** pour une chaîne de télévision étudiante, conçue pour donner la parole aux jeunes et promouvoir les talents étudiants dans le domaine des médias.

---

## 🏗️ **Architecture Technique**

### **🔧 Stack Technologique**
- **Frontend** : HTML5, CSS3, JavaScript, Bootstrap 5, EJS Templates
- **Backend** : Node.js 22, Express.js, SQLite3
- **Design** : Responsive Design, UI/UX moderne
- **Base de données** : SQLite3 avec tables structurées

### **📁 Structure du Site**
```
TELEX/
├── 📁 views/           # Templates EJS
│   ├── 📁 pages/      # Pages publiques
│   ├── 📁 admin/      # Interface administration
│   └── 📁 partials/   # Composants réutilisables
├── 📁 routes/          # Logique de navigation
├── 📁 config/         # Configuration base de données
├── 📁 public/         # Assets statiques
│   ├── 📁 css/        # Styles
│   ├── 📁 js/         # Scripts
│   └── 📁 images/     # Images et logos
└── 📁 database/       # Base de données SQLite
```
